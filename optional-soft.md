# optional-soft

## brew
- https://brew.sh/

## terminal translator 
1. `brew install node`,
2. `npm install fanyi`,
3. `which fanyi`,
4. `echo "alias t=/usr/local/bin/fanyi" >> ~/.bash_profile `,
5. `source ~/.zshrc` if you add `source ~/.bash_profile` in `~/.zshrc`.

## oh-my-zsh
- https://github.com/robbyrussell/oh-my-zsh,
